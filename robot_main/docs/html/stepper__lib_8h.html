Detailed Description

Function prototypes and public variables for stepper motor control.

This contains all functions and variables necessary for proper stpper motor control including velocity limits and motor specification (physical connections from MICRO-CONTROLLER to MOTOR, micro-step info etc.) The s_step() or s_stepAll() functions are executed periodically by TIM2 PeriodElapsedCallback event.
