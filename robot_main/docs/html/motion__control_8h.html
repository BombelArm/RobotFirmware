Detailed Description

Function prototypes and public variables for motion controll of the stepper motors.

This contains all functions and variables necessary for proper motion control including velocity,acceleration and position limits (of the joints). The m_control() and m_updatePositionAll() functions are executed periodically by TIM3 PeriodElapsedCallback event.
